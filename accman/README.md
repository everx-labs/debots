## Prerequisites

Install `everdev` and `make`.

Install `tonos-cli` (ver >= 0.13.1):

    everdev tonos-cli install

## How to compile

    make all

## How to deploy and run

Start Node SE:

    everdev se start

Then run:

    make deploy

## How to run

    make run

## How to debug

    make debug

## Deploy artifacts

`msig.addr` - file with deployed multisig address.

`msig.key` - file with multisig keys.
