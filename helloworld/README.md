# Hello World DeBot

Can be used as a template for new DeBots.

## How to build

    everdev sol compile helloDebot.sol

## How to deploy to node SE

Start node SE

    everdev se start

Deploy debot

    ./deploy_debot.sh

## How to run in node SE

    ./tonos-cli --url http://127.0.0.1 debot fetch <address>
