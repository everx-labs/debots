# Hello World DeBot

Can be used as a template for new DeBots.

## How to build

    tondev sol compile helloDebot.sol

## How to deploy to TON OS SE


Start TON OS SE

    tondev se start

Deploy debot

    ./deploy_debot.sh

## How to run in TON OS SE

    ./tonos-cli --url http://127.0.0.1 debot fetch <address>
